#  Application Flutter – Gestion d’Événements


#  Description du projet
 Cette application mobile développée avec Flutter
 permet la gestion complète des événements :
 création d’événements, réservation, authentification
 des utilisateurs et gestion des feedbacks.
 Elle s’adresse aussi bien aux organisateurs
 qu’aux utilisateurs.


#  Objectifs
 - Faciliter la création et la gestion des événements
 - Permettre aux utilisateurs de réserver des places
 - Gérer l’authentification (login / register)
 - Collecter les avis et feedbacks des participants
 - Offrir une interface simple et intuitive

#  Technologies utilisées
 - Flutter
 - Dart
 - SQLite (local database)
 - Architecture MVC / modulaire
 - Git & GitHub

#  Architecture du projet
 Le projet est structuré de manière modulaire pour assurer maintenabilité et évolutivité.

 #  Fonctionnalités principales
 - Authentification des utilisateurs
 - Création et modification des événements
 - Consultation de la liste des événements
 - Détails d’un événement
 - Réservation d’événements
 - Gestion des feedbacks et avis
 - Calendrier des événements
